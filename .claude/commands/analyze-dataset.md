# Analyze Dataset

Quick analysis of an AI training dataset.

## Instructions

Analyze the specified dataset using DataRecipe and provide a summary.

```bash
uv run datarecipe analyze $ARGUMENTS
```

After running, explain:
1. Dataset source and type
2. Generation method (synthetic ratio vs human ratio)
3. Teacher models used (if any)
4. Reproducibility score and what's missing
5. Estimated cost to recreate (if available)
