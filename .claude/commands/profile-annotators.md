# Profile Annotators

Generate annotator requirements and labor cost estimation for a dataset.

## Instructions

Generate an annotator profile for the specified dataset.

If no region is specified, default to "china".

```bash
uv run datarecipe profile $ARGUMENTS
```

After running, provide a summary:
1. Required skills and expertise level
2. Education and experience requirements
3. Team size recommendation
4. Estimated labor cost breakdown
5. Comparison with typical rates for this domain
